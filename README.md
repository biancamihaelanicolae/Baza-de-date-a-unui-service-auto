# Baza-de-date-a-unui-service-auto
Proiectul surprinde activitatea unui service auto care se ocupă cu reparațiile diverselor tipuri de autoturisme: mașini mici, SUV, dubiţe, oferind și servicii de schimb de piese depozitate în magazia serviceului (ex: ulei, plăcuțe de frână, filtre etc.) dacă acestea sunt necesare pe lângă alte probleme mecanice sau electrice.
